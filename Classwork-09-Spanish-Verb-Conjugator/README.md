This repository contains Python-based implementations, developed as part of the Programming course (Q2-2026). The objective of this assignment is to establish a professional development environment using Git for version control and GitHub for remote collaboration.

## Project Description

The included Python script (`spanish_verb_conjugator.py`) is a text-based tool that automatically conjugates regular Spanish verbs in the present tense. 

The program extracts the stem and the ending of the verb provided by the user and utilizes an `if/elif/else` decision flow to apply the correct conjugation rules for:
* **-ar** verbs (e.g., hablar)
* **-er** verbs (e.g., comer)
* **-ir** verbs (e.g., vivir)
