## Project Description

The included Python script (`numerical_integration.py`) is a mathematical tool designed to approximate the area under a curve (Numerical Integration) without needing analytical solving. 

The program utilizes a `for` loop iteration flow to continuously compute subintervals across four different mathematical methods:

* **Left Rectangle Method (LRM):** Uses the left edge of the subinterval to determine the rectangle's height.
* **Right Rectangle Method (RRM):** Uses the right edge of the subinterval to determine the rectangle's height.
* **Midpoint Rectangle Method (MPM):** Uses the center of the subinterval for a more balanced rectangle height.
* **Trapezoid Method (TRAP):** Averages the height of both edges of the subinterval to form a trapezoid, offering high accuracy for smooth curves.
